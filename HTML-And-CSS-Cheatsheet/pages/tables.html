HTML and CSS Cheatsheet

Tables

The Basics

In HTML, tables are created with the <table> tag. This then contains rows with the <tr> tag which then contain data with <td> and headers with <th>. Here's a basic example:

<table>
<tr>
<td>John Smith</td>
<td>42</td>
</tr>
</table>

Which produces:

John Smith 42


The Different Parts

Inside of a <table> tag, it's best practice to use the following tags to group together the different parts of the table:

  • <thead> contains the labels at the top

  • <tbody> contains the actual data of the table

  • And <tfoot> contains the footer with things like totals and averages if applicable.
